# Card com Tailwind CSS

Este projeto foi desenvolvido como parte do workshop **"Workshop Tailwind CSS: Dominando a Estilização Utilitária"**.

## 📚 Objetivo

O objetivo deste projeto foi praticar a criação de interfaces utilizando **Tailwind CSS**, explorando o conceito de **classes utilitárias** para estilização rápida e organizada.

Durante o workshop, foi demonstrado como construir um **card de produto responsivo**, aplicando estilos diretamente no HTML sem a necessidade de escrever CSS tradicional.

## 🧩 O que foi desenvolvido

Neste projeto foi criado:

* Um **card de produto**
* Estilização utilizando **Tailwind CSS**
* Estrutura simples em **HTML**
* Uso de **hover effects** e utilitários de layout

## 🛠 Tecnologias utilizadas

* HTML5
* Tailwind CSS (via CDN)
